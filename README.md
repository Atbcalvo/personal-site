# Personal Website (Local Only)

## Run Locally
```bash
npm install
npm run dev
```
Go to the localhost URL that Vite prints.

## Edit Content
Open `src/App.jsx` and edit the objects at the top:
- `PROFILE` — name, tagline, about, location, email.
- `LINKS` — GitHub/LinkedIn/Resume URLs.
- `PROJECTS` — title, blurb, tags, link.
- `EXPERIENCE` — org, role, time, bullets.

## Build (optional)
```bash
npm run build
npm run preview   # serve the production build locally
```
